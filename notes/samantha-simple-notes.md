# Samantha AI助手 - 最简化设计笔记

## 设计理念
- **专注核心**: 只保留3个核心功能
- **简单易用**: 界面简洁，功能明确
- **快速开发**: 6周完成MVP
- **成本低廉**: 最小化开发资源

## 核心功能 (3个)

### 1. 智能对话
- 基础文字问答
- GPT-3.5集成
- 简单对话历史

### 2. 语音交互
- 语音识别 (语音→文字)
- 语音合成 (文字→语音)
- 语音/文字切换

### 3. 情感回应
- 基础情感分析
- 情感化回复
- 简单用户偏好

## 技术架构 (简化)

### 前端
- **语言**: Kotlin
- **UI**: Jetpack Compose
- **存储**: Room (本地SQLite)
- **网络**: Retrofit

### 后端
- **语言**: Python
- **框架**: FastAPI
- **数据库**: SQLite
- **AI**: OpenAI GPT-3.5

### 架构图
```
Android App (Kotlin) ↔ Simple API (Python) ↔ OpenAI
```

## 数据库设计 (简化)

### 表结构
```sql
-- 对话表
CREATE TABLE conversations (
    id INTEGER PRIMARY KEY,
    user_id INTEGER,
    message TEXT,
    response TEXT,
    emotion TEXT,
    created_at TIMESTAMP
);
```

## API设计 (简化)

### 主要端点
- `POST /chat` - 发送消息
- `GET /history/{user_id}` - 获取历史
- `POST /preferences` - 保存偏好

### Android端
- 使用Jetpack Compose构建UI
- Room数据库存储本地数据
- Retrofit处理网络请求
- Hilt依赖注入

### 后端端
- FastAPI构建API服务
- SQLite存储数据
- OpenAI API集成
- 简单的情感分析

### 关键代码
- 主界面: ChatScreen
- 数据模型: Message, ChatUiState
- API调用: ChatRepository
- 语音处理: VoiceRepository

## 部署方案

### 本地开发
```bash
# 后端启动
cd backend
pip install -r requirements.txt
python main.py

# Android应用
cd android-app
./gradlew assembleDebug
```

### Docker部署
```bash
docker-compose up -d
```
